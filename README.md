# Arcelia Archive

A modern portfolio website built with Next.js, TypeScript, Tailwind CSS, and MySQL.

## Features

- Modern minimalist UI
- Archive system
- Admin dashboard
- Authentication
- Dynamic project pages (slug)
- Responsive layout

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- MySQL
- GSAP

## Installation

```bash
npm install
npm run dev
```
