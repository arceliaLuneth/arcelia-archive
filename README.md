# Arcelia Archive

Arcelia Archive is a personal portfolio website built to showcase projects, experiments, and development work in a clean and minimal interface.

The website includes a custom archive system with an admin dashboard for managing project content dynamically.

---

## Features

- Dynamic project archive
- Project detail pages with slug routing
- Admin authentication
- Archive management (Create, Read, Update, Delete)
- Responsive layout
- Modern minimalist UI
- Smooth page animations using GSAP

---

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- GSAP
- MySQL
- Node.js

---

## Installation

Clone this repository.

```bash
git clone https://github.com/YOUR_USERNAME/arcelia-archive.git
```

Install dependencies.

```bash
npm install
```

Create an environment file.

```env
DB_HOST=
DB_USER=
DB_PASSWORD=
DB_NAME=
```

Run the development server.

```bash
npm run dev
```

Open:

```
http://localhost:3000
```
